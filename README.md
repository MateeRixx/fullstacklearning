# 🛒 Simple Ecommerce Cart

A **basic ecommerce cart project** built using **Vanilla JavaScript, HTML, and CSS**.

This project demonstrates how a simple shopping cart works — users can browse products, add them to a cart, view the total price, and checkout.

---

# ✨ Features

- 🛍️ Product listing with price  
- ➕ Add products to cart  
- 🧾 View cart items and total price  
- 🧹 Checkout button to clear the cart

---

# 💡 Example Usage

1. Click **Add to Cart** on any product  
2. The product appears in the cart  
3. The **total price updates automatically**  
4. Click **Checkout** to clear the cart
